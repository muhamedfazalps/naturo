### Findings
1. **JSON tree is nested (children arrays), not flat** — scripters must write recursive traversal. The `elements` key doesn't exist; data is in root + `children`. This is a discoverability issue but not a bug.
2. **JSON Unicode handling is correct** — Chinese chars are proper `\uXXXX` escapes, parseable by Python `json.loads()`. Console display may garble but data is sound.
3. **Exit codes correct throughout pipeline** — `set -e` bash scripts work correctly.
4. **stderr/stdout separation is clean in JSON mode** — no leakage to stderr (#783 not reproduced in these scenarios).
5. **Full automation pipeline works**: launch → JSON parse → extract refs → click sequence → verify display → quit. All passed.
